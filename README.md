# Intruder Alarm PCB

## Project Description
This project is an Intruder Alarm system designed using KiCad. The project includes schematic design and PCB layout for detecting unauthorized entry and triggering an alarm.

## Features
- Intrusion detection
- Alarm activation
- Custom PCB design
- Designed using KiCad

## Software Used
- KiCad

## Project Status
PCB designed successfully in KiCad. Hardware implementation and testing are planned for future development.
